# /sync-docs

`op-cache.sh` の現在の実装を読み、以下の4ファイルを実装と一致するよう更新してください。

## 対象ファイル

1. **SPEC.md** — サブコマンド仕様・フロー・データ形式・内部処理の詳細が実装と一致しているか確認し、乖離があれば修正する
2. **CLAUDE.md** — コマンド一覧・アーキテクチャ説明・注意事項が実装と一致しているか確認し、乖離があれば修正する
3. **README.md** — 英語版。使い方・コマンド一覧・設定が実装と一致しているか確認し、乖離があれば修正する
4. **README.ja.md** — 日本語版。README.md と同内容で一致しているか確認し、乖離があれば修正する

## 手順

1. `op-cache.sh` を読んで現在の実装を把握する（サブコマンド・オプション・出力メッセージ・環境変数）
2. 各ドキュメントを読んで実装との乖離を洗い出す
3. 乖離があるファイルのみ更新する（変更不要なファイルはスキップ）
4. 何を変更したか（または変更なしだったか）を簡潔に報告する

## 注意事項

- 実装に存在しない機能をドキュメントに書かない
- ドキュメントに書かれていて実装にない場合は、実装を正として削除する
- 出力メッセージの文言（日本語含む）は実装のものをそのまま使う
- README には実装の詳細すぎる内部情報は書かない（SPEC.md の役割）
